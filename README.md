# SehatSetu – Rural Healthcare Platform

## Overview
SehatSetu is a full-stack web application designed to improve healthcare accessibility in rural areas by integrating telemedicine, health education, and pharmacy services.

## Features
- User Registration and Login (Backend integrated)
- Online Doctor Consultation
- AI Health Assistant
- Health & Hygiene Awareness Section
- Pharmacy Access Simulation

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js

## How It Works
Users can register/login, consult doctors by entering symptoms, and receive basic guidance. The backend handles user data and consultation logic.

## Note
The frontend is deployed using GitHub Pages.
The backend runs locally using Node.js server (localhost).
